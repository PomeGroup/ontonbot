# Check Status Component Images

To complete the implementation of the Check Status page, please add the following NFT images to your storage or public folder:

1. `platinum-nft.png` - An image representing Platinum level NFTs
2. `gold-nft.png` - An image representing Gold level NFTs
3. `silver-nft.png` - An image representing Silver level NFTs
4. `bronze-nft.png` - An image representing Bronze level NFTs

These images should be 56x56px or larger, and will be used in the NFT Ownership section of the page.

## URL Locations

If using a storage service, the full URLs are expected to be:

- https://storage.onton.live/ontonimage/platinum-nft.png
- https://storage.onton.live/ontonimage/gold-nft.png
- https://storage.onton.live/ontonimage/silver-nft.png
- https://storage.onton.live/ontonimage/bronze-nft.png

Alternatively, you can place these images in the public folder at:

- `/public/images/platinum-nft.png`
- `/public/images/gold-nft.png`
- `/public/images/silver-nft.png`
- `/public/images/bronze-nft.png`

And update the image paths in the component accordingly.
